## Getting Started

Clone from GitHub 
```bash
git clone https://github.com/themewagon/Resume-Nextjs.git
```

Install dependencies:
```bash
npm install
```

Run development server:
```bash
npm run dev
```

Build for production:
```bash
npm run build
```
